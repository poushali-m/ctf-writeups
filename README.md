# CTF Write-ups & Security Lab Reports

A collection of penetration testing lab reports and CTF challenge write-ups completed 
as part of advanced cyber security training. Each write-up documents the full attack 
chain: reconnaissance, enumeration, exploitation, privilege escalation, and lessons learned.

## Challenges

| Challenge | Key Techniques | Tools Used |
|-----------|---------------|------------|
| Corrosion | Tomcat exploitation, ZIP cracking, Python module hijacking | Metasploit, fcrackzip, John the Ripper |
| Mission Pumpkin | FTP enumeration, Base64 decoding, sudo misconfiguration | nmap, gobuster, ssh |
| WestWild | SMB enumeration, lateral movement, privilege escalation | enum4linux, smbclient, Metasploit |
| Footprinting & Recon | TCP/UDP scanning, OS fingerprinting | Nmap, arp-scan |
| Network Scanning | Null/FIN scan analysis, firewall detection | Nmap |

## Skills Demonstrated

- Network reconnaissance and service enumeration
- Credential extraction and password cracking
- Web application enumeration and directory brute-forcing
- Metasploit framework usage
- Linux privilege escalation techniques
- MITRE ATT&CK technique mapping
- Incident documentation and reporting

## Environment

All activities were performed in isolated, controlled lab environments 
using Kali Linux against intentionally vulnerable target machines.
